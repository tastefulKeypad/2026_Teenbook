# Проект "Я и мой быт"

Этот раздел посвящен практическим аспектам подростковой повседневности дома: личному пространству, бытовой безопасности, обязанностям, технике и домашним животным.

Материалы оформлены как связанный набор концептов, текстов и артефактов для дальнейшей интеграции в общую энциклопедию TeenBook.

## Цель раздела

Собрать структурированную базу знаний, которая:

- объясняет бытовые темы простым языком
- показывает связи между понятиями из разных бытовых направлений
- использует данные из Wikidata и SPARQL
- поддерживает формат графа знаний с перекрестными ссылками

## Предметная область

Тема раздела:

**"Я и мой быт"**

Подразделы:

- `my_space`
- `household_security`
- `household_responsibilities`
- `household_appliances`
- `household_pets`

## Структура раздела

### WORK

В рабочих подпапках содержатся:

- `concepts.json` и списки терминов
- SPARQL-запросы и результаты выгрузки
- Python-скрипты обработки
- онтологии и схемы (в том числе Mermaid/PlantUML)
- вспомогательные данные и промежуточные артефакты

### WEB

В соответствующих подпапках `WEB/me_and_my_everyday_life` размещаются:

- финальные Markdown-статьи
- материалы с перекрестными ссылками
- итоговые тексты для публикации

## Перекрестные ссылки

Для раздела предусмотрена расстановка ссылок между понятиями (вручную и/или скриптами на уровне проекта):

- ключевые термины берутся из `concepts.json`
- целевые ссылки добавляются в статьи подразделов
- при наличии скриптов автоматизации ссылки можно расставлять пакетно
- связи можно поддерживать как внутри подраздела, так и между подразделами

## Работа с Wikidata

В подразделах используются SPARQL-запросы к Wikidata для:

- подбора сущностей и Q-id
- извлечения описаний и связанных понятий
- экспорта результатов в JSON для дальнейшего использования

Пример реализованного пайплайна есть в:

- `household_pets/scripts/wikidata_query.py`
- `household_pets/data/wikidata_export.json`

## Онтология и визуализация

Концептуальная модель раздела включает:

- ключевые сущности и их отношения
- иерархические и горизонтальные связи
- визуализацию связей в формате диаграмм

Для `household_pets` добавлена Mermaid-схема:

- `household_pets/images/Teen Care and Pet-2026-03-20-200248.mmd`

## Используемые технологии

- **Markdown** - тексты и документация
- **Python** - автоматизация обработки
- **Wikidata + SPARQL** - структурированные знания
- **JSON** - хранение концептов и результатов
- **Mermaid / PlantUML** - визуализация онтологии

## Результат

В разделе сформирован набор материалов, который можно напрямую использовать для сборки финальной энциклопедии:

- структурированные концепты
- скрипты получения знаний

- база для межраздельной интеграции ссылок
