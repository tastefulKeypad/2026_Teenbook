## Что я делал

Выбрал тему про безопасность дома. Написал .md статьи в директорию WEB, разобрался с WikiData и SPARQL, построил онтологию из понятий, которые удалось найти в базе.

## Понятия и связи между ними

Я выделил 12 понятий, которые описывают безопасность дома и подростка.

- **Подросток** - ключевая сущность
- **Незнакомец** - потенциально опасный человек, если подросток один дома
- **Безопасность** - ощущение надежности и сохранности
- **Опасность** - потенциальные угрозы
- **Одиночество** - когда подросток один дома

Не забудь выключить их перед уходом из дома:
- **Утюг**
- **Плита**
- **Электроприборы**

А так же другие важные сущности из этого раздела:
- **Аптечка**
- **Труба** - например, прорванная труба
- **Запах** - например, запах газа

## SPARQL-запрос

Использовал один запрос — на получение понятий и их описаний из WikiData:

```sparql
QUERY = """
SELECT ?item ?itemLabel ?description WHERE {
  VALUES ?item {
    wd:Q131774      # adolescence (подросток)
    wd:Q13197964    # stranger (незнакомец)
    wd:Q10566551    # safety (безопасность)
    wd:Q483634      # clothes iron (утюг)
    wd:Q182995      # kitchen stove (плита)
    wd:Q12069731    # space heater (обогреватель)
    wd:Q2425052     # electrical appliance (электроприбор)
    wd:Q815867      # first aid kit (аптечка)
    wd:Q41551       # pipe (труба)
    wd:Q485537      # odor (запах)
    wd:Q1132455     # hazard (опасность)
    wd:Q6010868     # solitude (одиночество)
  }

  SERVICE wikibase:label { bd:serviceParam wikibase:language "ru,en". }

  OPTIONAL {
    ?item schema:description ?description .
    FILTER(LANG(?description) = "ru")
  }
}
"""
```

## Как шла работа

Зашел Wikidata и искал там понятия похожие на те темы, по которым писались статьи в папке WEB.

Сделал простую mermaid markdown (.mmd) схему. Оказалось довольно просто и удобно!

## Личные ощущения

Больше всего времени ушло не на код, а на то, чтобы найти правильные понятия в WikiData. База большая, но социальная тематика там покрыта слабо — многие очевидные связи просто отсутствуют. Так же очень часто всплывали ссылки на различные медиа произведения. По запросу "незнакомец" примерно 40 запросов были медиа произведениями, тем самым усложнив мне поиск...
